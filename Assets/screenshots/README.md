# Release screenshots

Use these folders for per-version screenshots used in store listings and release notes.

## Suggested naming
- `vX.Y.Z-description.png`
- Example: `v2.0.0-home-map.png`

## Folders
- `android/` - Play Store phone screenshots (Drizzle)
- `windows/` - Windows app screenshots (README/release notes)
- `ios-iphone/` - App Store iPhone screenshots (iDrizzle)
- `ios-ipad/` - App Store iPad screenshots (iDrizzle)
